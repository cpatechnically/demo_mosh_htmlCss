Data list (AKA suggestion list) in forms

Form Data List

Can assing a numeric value to each option via the "data-value" form property. This is how a response could be associated to a value on the server. Users can override a data list item in the form input because it is a "suggestion"

Dropdown list in forms

Dropdown List

Assign a numeric value to each option via the "value" (NOT DATA-VALUE) form property. This is how a response could be associated to a value on the server. Users CANT override a dropdown list
Select a "default selected option" by applying the keyword "selected" to a single value from the list.
